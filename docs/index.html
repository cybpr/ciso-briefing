ACT NOW CVE-2026-33824 — Windows IKE CVSS 9.8 RCE (April 2026 Patch Tuesday)
RPS: 9.09 S:9.8 · E:8 · X:9
📰 Source: CrowdStrike Patch Tuesday Analysis 🗓 Published: April 15, 2026 🧩 MITRE ATT&CK: T1190 (Exploit Public-Facing App) · T1068 (Exploit for Privilege Escalation) · T1210 (Exploit Remote Services)

CVE-2026-33824 is a Critical remote code execution vulnerability affecting Windows Internet Key Exchange (IKE) Service Extensions with a CVSS score of 9.8. It allows unauthenticated remote attackers to execute arbitrary code by exploiting a double free flaw (CWE-415) in the Windows IKE Extension. No user interaction is required and attack complexity is low — an unauthenticated attacker can exploit the flaw by sending specially crafted packets to a Windows machine with IKE version 2 enabled. The highest rated vulnerability in April's batch, it could allow a threat actor to run remote code on a system unless it is patched immediately, with attack complexity low and no user privileges required, affecting Windows Server 2025, 2022, 2019, 2016, and Windows 10/11 x64, 32-bit, and ARM systems. Admins unable to immediately install the update should either block inbound UDP ports 500 and 4500 for systems not using IKE, or restrict inbound traffic on those ports to known peer addresses only. While Microsoft has marked the flaw as "exploitation less likely," historical exploits and proof-of-concept code for IKE exist, meaning motivated threat actors may be able to weaponize this vulnerability.

ACT NOW Axios npm Supply Chain Attack — North Korea (UNC1069) Deploys Cross-Platform RAT
RPS: 8.25 S:8 · E:8 · X:10
📰 Source: Google Cloud Threat Intelligence (GTIG) · Microsoft Security Blog 🗓 Published: March 31 – April 1, 2026 🧩 MITRE ATT&CK: T1195.001 (Supply Chain: Compromise Software Dependencies) · T1059.007 (JS/Script Execution) · T1219 (Remote Access Tools) · T1078 (Valid Accounts) · T1552 (Unsecured Credentials)

Google Threat Intelligence Group (GTIG) is tracking an active software supply chain attack targeting the popular npm package "axios." Between March 31, 2026, 00:21 and 03:20 UTC, an attacker introduced a malicious dependency named "plain-crypto-js" into axios npm releases versions 1.14.1 and 0.30.4 — packages with over 100 million and 83 million weekly downloads respectively. The malicious dependency is an obfuscated dropper that deploys the WAVESHAPER.V2 backdoor across Windows, macOS, and Linux. GTIG attributes this activity to UNC1069, a financially motivated North Korea-nexus threat actor active since at least 2018. The attacker compromised a maintainer account and published backdoored versions that delivered a cross-platform Remote Access Trojan to macOS, Windows, and Linux systems through a malicious postinstall hook. Users who installed Axios version 1.14.1 or 0.30.4 should rotate their secrets and credentials immediately and downgrade to a safe version (1.14.0 or 0.30.3), and should also disable auto-updates for Axios npm packages since the malicious payload includes a hook that will continue to attempt to update. OpenAI revoked its macOS app-signing certificate after discovering that a malicious version of the Axios npm package was pulled into its workflow — the supply chain attack, attributed to North Korean state-sponsored actors, injected credential-stealing malware, and older macOS apps will stop working after May 8, 2026.
