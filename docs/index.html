Microsoft addressed 164 vulnerabilities in its April 2026 security update release — double the number patched in March — including one exploited zero-day, one previously disclosed zero-day, and eight Critical-rated vulnerabilities. The headline issue is CVE-2026-32201, a spoofing vulnerability in Microsoft SharePoint Server rated CVSS 6.5 "Important," where improper input validation allows an unauthorized attacker to perform spoofing over a network, potentially enabling sensitive information access and data manipulation. CISA's KEV catalog already includes ten SharePoint flaws; CVE-2026-32201 was added immediately and federal agencies were instructed to patch by April 28. Additionally, CVE-2026-33824 is a Critical Windows IKE RCE rated CVSS 9.8, exploitable by unauthenticated remote attackers via a double-free flaw, requiring no user interaction and allowing arbitrary code execution on systems with IKEv2 enabled. CVE-2026-33825 (BlueHammer) was observed in the wild as part of a vulnerability chaining attempt to deploy BeigeBurrow, a tunneling or proxy agent, as of April 21.

The intrusion began with a Lumma Stealer malware infection at Context.ai in approximately February 2026; attackers leveraged a compromise of Context.ai's Google Workspace OAuth tokens to gain a foothold into Vercel's internal systems, exposing environment variables for an undisclosed but reportedly limited subset of customer projects. The incident originated with a compromise of Context.ai, a third-party AI tool used by a Vercel employee; the attacker used that access to take over the employee's individual Vercel Google Workspace account, then pivoted into a Vercel environment and maneuvered through systems to enumerate and decrypt non-sensitive environment variables. Following the breach, threat actors posted the leaked Vercel database for sale at $2M on BreachForums. From a compromised deployment platform, an attacker may be able to read or modify build settings, add malicious build steps, trigger deployments, and extract environment variables — turning a third-party AI tool compromise into an end-to-end supply-chain attack.

CISA added CVE-2026-20133 to its Known Exploited Vulnerabilities catalog "based on evidence of active exploitation" and ordered Federal Civilian Executive Branch agencies to secure their networks by Friday, April 24. CVE-2026-20133 (CWE-200 – Sensitive Information Exposure) allows remote, unauthenticated attackers to view sensitive information on affected systems. Catalyst SD-WAN Manager is a centralized tool used to monitor and manage up to 6,000 Cisco SD-WAN devices, making a successful exploit a potential gateway to mass enterprise network intrusion. While ransomware involvement is currently listed as "unknown," the exploitation of SD-WAN management platforms has historically preceded large-scale network intrusions; CISA has issued Emergency Directive 26-03 along with dedicated Hunt & Hardening Guidance for Cisco SD-WAN Devices, underscoring the threat's severity.
