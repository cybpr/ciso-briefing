📅 Apr 14, 2026 (Patch Tuesday) 🔗 CrowdStrike Analysis  |  HivePro Advisory 🎯 MITRE ATT&CK: T1210 (Exploitation of Remote Services), T1595.002 (Vulnerability Scanning), T1021 (Remote Services), T1570 (Lateral Tool Transfer)

CVE-2026-33824 (Windows IKE Service Extensions, CVSS 9.8) and CVE-2026-33827 (Windows TCP/IP, CVSS 8.1) are both unauthenticated, network-exploitable RCE vulnerabilities with wormable characteristics; the IKE flaw targets systems with IKE v2 enabled, while the TCP/IP vulnerability affects IPv6/IPsec environments via a race condition. Unauthenticated RCE against modern Windows assets is relatively rare; because IKE provides secure tunnel negotiation services for VPNs, it is necessarily exposed to untrusted networks and reachable in a pre-authorization context. CVE-2026-33824 received a CVSSv3 score of 9.8, was rated critical, and can be exploited by an unauthenticated attacker by sending crafted packets to a target with IKE version 2 enabled. Interim mitigation includes blocking inbound traffic on UDP ports 500 and 4500 if IKE is not in use. Immediate patch deployment is the primary remediation path; all Windows endpoints with IPv6 or IPSec enabled are in scope.
